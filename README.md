# Measurement Tracker (Java)

This project is a Java desktop application built using Swing and SQLite.

## Features
- Add body measurements (date, weight, waist, chest, notes)
- View saved measurements in a table
- Data stored locally using SQLite

## Technologies
- Java
- Swing (GUI)
- SQLite (database)
- IntelliJ IDEA

## OOP Principles Used
- Encapsulation (data access separated via DAO)
- Object-oriented design with model classes
